# project-kelompok-1
Repositori ini berisi kode sumber untuk aplikasi mobile (Android/iOS) yang bertujuan menyediakan informasi komprehensif mengenai kampus. Aplikasi ini dirancang untuk menjadi sumber daya utama bagi calon mahasiswa, mahasiswa aktif, dan masyarakat umum untuk mendapatkan detail penting seputar institusi.
